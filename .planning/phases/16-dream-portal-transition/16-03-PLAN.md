<action>
1. **Modify wireShaders.js** -- add a `uWireTransitionAlpha` uniform to the fragment shader for transition-controlled visibility.

   Add to the fragment shader uniforms (after `uWirePulse` on line 29):
   ```glsl
   uniform float uWireTransitionAlpha;  // 1.0 = normal, 0.0 = hidden (Phase 16 transition control)
   ```

   Modify the alpha computation (line 37). Change:
   ```glsl
   float alpha = vWireAlpha * pulse * 0.4;
   ```
   To:
   ```glsl
   float alpha = vWireAlpha * pulse * 0.4 * uWireTransitionAlpha;
   ```

   This allows external controllers (DreamTransition) to fade wires to zero during dissolve/tunnel and flash them briefly during the void.

2. **Update ParticleMemoryField.jsx** -- add `uWireTransitionAlpha` to the wireUniforms ref (after `uWirePulse` on line 30):
   ```js
   uWireTransitionAlpha: { value: 1.0 },  // Phase 16: transition visibility control
   ```

3. **Expose wireUniforms via the existing useImperativeHandle** -- currently (line 34) only `uniforms` (particle uniforms) is exposed. Expand to include wire uniforms:
   ```js
   useImperativeHandle(ref, () => ({
     uniforms: uniforms.current,
     wireUniforms: wireUniforms.current,
   }));
   ```
   This allows DreamTransition to access `fieldRef.current.wireUniforms.uWireTransitionAlpha.value`.
</action>

<action>
Modify `src/components/DreamTransition.jsx` to add wire filament flash effects to both entry and exit timelines.

1. **In buildDreamEntryTimeline** -- add wire fade and flash bursts:

   After the dissolve label, fade wires to zero:
   ```js
   // Fade wires during dissolve (they'd appear at wrong positions when scattered)
   const wireUniforms = fieldRef.current?.wireUniforms;
   if (wireUniforms) {
     tl.to(wireUniforms.uWireTransitionAlpha, {
       value: 0,
       duration: 1.0,
       ease: 'power2.in',
     }, 'dissolve');
   }
   ```

   During the tunnel void label, add intermittent flash bursts:
   ```js
   // Filament flashes during tunnel void -- brief luminous wire structures appear and snap away
   if (wireUniforms) {
     const flashTimes = [0.2, 0.5, 0.75];  // seconds into tunnel phase
     flashTimes.forEach(t => {
       tl.to(wireUniforms.uWireTransitionAlpha, {
         value: 0.35, duration: 0.06, ease: 'none',
       }, `tunnel+=${t}`);
       tl.to(wireUniforms.uWireTransitionAlpha, {
         value: 0, duration: 0.12, ease: 'power2.out',
       }, `tunnel+=${t + 0.06}`);
     });
   }
   ```

   At the reform label, restore wire visibility gradually:
   ```js
   // Restore wires as particles reform (only visible when morph > 0.7)
   if (wireUniforms) {
     tl.to(wireUniforms.uWireTransitionAlpha, {
       value: 1.0,
       duration: 1.0,
       ease: 'power2.out',
     }, 'reform+=1.0');  // delayed start -- wires appear after particles mostly reformed
   }
   ```

2. **In buildDreamExitTimeline** -- mirror the wire effects:
   - At dissolve start: fade wires to 0 over 1.0s
   - During tunnel void: same flash burst pattern
   - No wire restore needed (component unmounts after navigation)

3. **Guard all wireUniforms access** with null checks since wireData may be null (simplified tier has no wires).
</action>
