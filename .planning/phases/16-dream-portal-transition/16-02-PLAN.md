# Plan 16-02: DreamTransition Component + GSAP Entry/Exit Timelines

<objective>
Create DreamTransition.jsx — a GSAP-based transition orchestrator that drives the 3-phase dream portal sequence (dissolve → tunnel void → reform) by animating uMorphProgress, uMorphStagger, and camera FOV on the particle field. Add tunnel mode to FlightCamera (auto-advance, FOV override skip). Integrate into CapsuleShell as the transition system for particle-memory scenes, replacing PortalVFX. Store departure state in sessionStorage for intentional return transitions.
</objective>

<task id="16-02-01" title="Add tunnel mode to FlightCamera">
Add `setTunnelMode` and `getCamera` to FlightCamera's useImperativeHandle. When tunnel mode is active, skip the FOV bell-curve computation (let GSAP tween FOV externally) and auto-advance progress at a constant rate.
</task>

<task id="16-02-02" title="Expose fieldRef and flightCameraRef from ParticleFieldRenderer">
ParticleFieldRenderer already exposes these via useImperativeHandle. Verify the ref shape and add any missing accessors needed by DreamTransition (getCamera from FlightCamera, uniforms/wireUniforms from ParticleMemoryField).
</task>

<task id="16-02-03" title="Create DreamTransition.jsx with GSAP 3-phase timelines">
Create the DreamTransition component with:
- `buildDreamEntryTimeline(fieldRef, cameraRef, scene)` — dissolve (morph 1→0, stagger ramp), tunnel void (auto-advance camera, FOV narrow), reform (morph 0→1 with stagger, FOV restore)
- `buildDreamExitTimeline(fieldRef, cameraRef, scene)` — reverse grammar: dissolve (morph 1→0), tunnel void, then navigate away
- `useDreamTransition` hook that manages entry/exit state and sessionStorage
</task>

<task id="16-02-04" title="Integrate DreamTransition into CapsuleShell for particle-memory scenes">
Replace PortalVFX exit sequence for particle-memory scenes with DreamTransition. Store departure state in sessionStorage on entry. Use dream exit timeline on back button instead of PortalVFX phase sequence.
</task>

<task id="16-02-05" title="Build verification">
Run `npx vite build` to verify no import errors or shader issues. Verify DreamTransition exports are correct and CapsuleShell compiles.
</task>
