**Create `src/utils/streaks.js`:**
- localStorage key: `jarowe_streak`
- Schema: `{ count: number, lastVisit: string (YYYY-MM-DD), freezeAvailable: boolean, freezeUsed: boolean }`
- `getStreakData()`: Read and parse from localStorage. Return defaults `{ count: 0, lastVisit: null, freezeAvailable: true, freezeUsed: false }` if missing or corrupt.
- `checkStreak()`: Called once per page load. Get today's date as `YYYY-MM-DD` string. Compare to `lastVisit`:
  - Same day: No change, return `{ count, isNew: false, milestone: null }`
  - Yesterday: Increment count, update lastVisit, return `{ count, isNew: true, milestone: null or milestone number }`
  - 2+ days gap: If `freezeAvailable && !freezeUsed`, use freeze (set freezeUsed=true, freezeAvailable=false, keep count, update lastVisit). Otherwise reset count to 1.
  - No previous visit: Set count=1, lastVisit=today.
  - Milestone detection: After increment, check if count is 3, 7, 14, or 30. If 30, also recharge freeze (freezeAvailable=true, freezeUsed=false).
  - Save updated data to localStorage.
  - Return `{ count, isNew: boolean, milestone: number|null, frozeUsed: boolean }`
- `useStreakFreeze()`: Manually use the freeze (for future UI). Sets freezeAvailable=false, freezeUsed=true.
- Helper `getTodayStr()`: Returns `YYYY-MM-DD` string for today in local timezone.
- Helper `getYesterdayStr()`: Returns `YYYY-MM-DD` string for yesterday.
- Helper `daysBetween(dateStr1, dateStr2)`: Returns integer days between two YYYY-MM-DD strings.

**Create `api/og.js`:**
- Vercel Function (Node.js runtime, NOT Edge) using `@vercel/og` ImageResponse.
- Export default GET handler. Read `route` query param from URL.
- 4 route templates (all 1200x630px):
  1. **homepage** (default / route='/'): Dark gradient background (#080810 to #1a1a2e), "jarowe.com" in large white text, subtitle "The most alive personal world on the internet", subtle star particles, Starseed gold accent line.
  2. **constellation** (route='/constellation'): Deep space dark background, "Constellation" title, subtitle "Explore a 3D map of life moments", scattered dot pattern suggesting nodes.
  3. **games** (route contains 'game'): Playful gradient (purple to blue), "Mini Games" title, game controller emoji, "Play 20+ mini-games" subtitle.
  4. **starseed** (route starts with '/starseed'): Warm dark background (#080810), Starseed gold (#dbb978) accents, "Starseed" title, "Creative solutions & labs" subtitle.
- All templates use system fonts (no custom font loading needed for v1): `font-family: 'system-ui, -apple-system, sans-serif'`.
- JSX layout via Satori (built into @vercel/og): Use `display: 'flex'` with flexbox layout in the JSX.
- Return `new ImageResponse(jsx, { width: 1200, height: 630 })`.
- Add `export const config = { runtime: 'nodejs' }` at the top level (NOT edge — @vercel/og works on Node.js per user decision).
- Handle unknown routes by falling back to the homepage template.
- Cache: Set `Cache-Control: public, max-age=86400, s-maxage=86400` header for CDN caching.
