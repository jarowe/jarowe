**Phase:** 14-particle-field-core

## Goal

Add spatial-hash wire connections (LineSegments) to the modular particle memory component architecture and verify tier adaptation correctly gates wires behind full tier, simplified tier renders without wires/bloom, and parallax fallback shows CSS particle dots.

## Context
